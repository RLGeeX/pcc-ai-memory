# Python Commands

- `mkvenv` - Create a virtual environment for the project.
- `pip install -r requirements.txt` - Install project dependencies in the active virtual environment.
- `which python` - Verify the active virtual environment by checking the Python executable path.
- `pip list` - List installed packages to verify dependencies.
- `pytest` - Run the project's test suite using pytest.
- `ruff check .` - Lint the codebase for style and quality issues using ruff.
- `ruff format .` - Format the codebase according to style guidelines using ruff.
- `mypy .` - Perform static type checking on the codebase using mypy.

## Note
Run `mkvenv` and `pip install -r requirements.txt` after cloning the repository. Add new commands here as discovered.
